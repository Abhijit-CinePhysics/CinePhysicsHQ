CinePhysics Practice Worksheet

Class: XI
Chapter: Units and Measurements
Worksheet Type: Practice
Difficulty Level: Mixed

Learning Outcomes

  1. Understand the need for measurement and standard units.
  2. Identify physical quantities and SI units.
  3. Differentiate between fundamental and derived quantities.
  4. Apply concepts of significant figures and measurement errors.
  5. Use dimensional formulae and dimensional analysis correctly.

Instructions

  1. Answer all questions.
  2. Use proper SI units.
  3. Show all calculations in numerical questions.
  4. Apply significant figure rules wherever required.
  5. Use dimensional formulae correctly.

Section A: Multiple Choice Questions

  1. Why is measurement necessary in science?
    1. To compare quantities accurately
    2. To avoid calculations
    3. To replace experiments
    4. To increase observations
  2. Which of the following is a physical quantity?
    1. Beauty
    2. Mass
    3. Honesty
    4. Intelligence
  3. Which system uses metre, kilogram and second as fundamental units?
    1. CGS
    2. FPS
    3. MKS
    4. British
  4. The SI unit of electric current is:
    1. Volt
    2. Ohm
    3. Ampere
    4. Watt
  5. Which of the following is a derived quantity?
    1. Length
    2. Mass
    3. Density
    4. Time
  6. The dimensional formula of force is:
    1. [M L T-2]
    2. [M L2 T-2]
    3. [M L-1 T-2]
    4. [M T-2]
  7. How many significant figures are present in 0.00560?
    1. 2
    2. 3
    3. 4
    4. 5
  8. The SI unit of force is:
    1. Joule
    2. Newton
    3. Pascal
    4. Watt
  9. Which error can be reduced by taking repeated measurements?
    1. Random error
    2. Gross error
    3. Systematic error
    4. Instrument defect
  10. The dimensional formula of density is:
    1. [M L-3]
    2. [M L3]
    3. [M-1 L-3]
    4. [M L-2]
  11. Which application uses dimensional analysis?
    1. Checking equation consistency
    2. Measuring temperature
    3. Drawing graphs
    4. Calculating averages only
  12. Which of the following has the dimensions of pressure?
    1. [M L T-2]
    2. [M L-1 T-2]
    3. [M L2 T-2]
    4. [M L-2]

Section B: Assertion Reason

  1. Assertion: SI units are internationally accepted. Reason: They provide uniform standards for measurements.
  2. Assertion: Area is a derived quantity. Reason: Area is obtained from multiplication of two lengths.
  3. Assertion: Random errors can be minimized by repeated observations. Reason: Averaging reduces random fluctuations.
  4. Assertion: Dimensional analysis can determine numerical constants in equations. Reason: Dimensions alone provide numerical values of constants.

  1. Discuss the need for measurement and explain why standard systems of units are necessary for scientific communication.
  2. Explain different types of errors in measurement and methods to reduce them.
  3. Define dimensions and dimensional formulae. Derive the dimensional formula of force.

Section F: Numericals

  1. A student measures the length of a rod five times and obtains 20.1 cm, 20.2 cm, 20.3 cm, 20.2 cm and 20.2 cm. Calculate the mean length.
  2. Round 0.0067834 to three significant figures.
  3. Find the dimensional formula of work done using:
    Work = Force × Displacement
  4. The measured value of a quantity is 25.6 cm and the true value is 25.0 cm. Calculate the absolute error.
  5. A force of 50 N acts on an area of 10 m2. Calculate the pressure and write its SI unit.
