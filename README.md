# 🚀 CinePhysicsHQ

**Interactive Physics Simulations, Virtual Labs, Educational Tools, and STEM Learning Resources**

CinePhysicsHQ is an open educational initiative focused on making Physics visual, interactive, and engaging through simulations, virtual laboratories, worksheets, and educational technology.

🌐 Website: https://abhijit-cinephysics.github.io/CinePhysics/

---

## 📚 About

Physics is best understood when concepts can be visualized and explored.

CinePhysicsHQ provides:

- Interactive Physics Simulations
- Virtual Physics Laboratories
- Dimensional Analysis Tools
- Buckingham Pi Theorem Calculator
- Error Analysis Laboratory
- Educational Worksheets
- Question Banks
- STEM Learning Resources

Designed for students, educators, and lifelong learners.

---

## 🔬 Featured Physics Labs

### 📏 Dimensional Analysis Lab

Check whether any physical equation is dimensionally correct.

Features:

- Equation Verification
- Dimensional Homogeneity Check
- Step-by-Step Dimensional Breakdown
- Unit Consistency Analysis
- Instant Results

Perfect for:

- CBSE Class 11 Physics
- JEE Main
- JEE Advanced
- NEET
- Engineering Physics

---

### π Buckingham Pi Theorem Lab

Generate dimensionless groups automatically using Buckingham Pi Theorem.

Features:

- Variable Input System
- Dimensional Matrix Generation
- π-Term Calculation
- Dimensionless Verification
- Research and Project Applications

Ideal for:

- Fluid Mechanics
- Aerodynamics
- Experimental Physics
- Engineering Analysis

---

### 📊 Error Analysis Laboratory

Learn uncertainty and measurement analysis through interactive experiments.

Features:

- Absolute Error
- Relative Error
- Percentage Error
- Propagation of Uncertainty
- Statistical Analysis
- Monte Carlo Simulations

---

### ⚡ Lorentz Force Simulation

Visualize charged particle motion in electric and magnetic fields.

Explore:

- Magnetic Force
- Velocity Effects
- Charge Polarity
- Circular Motion
- Helical Motion

---

### 🧲 Magnetic Force Between Parallel Conductors

Interactive simulation demonstrating Ampere's Force Law.

Features:

- Adjustable Currents
- Force Visualization
- Field Representation
- Real-Time Interaction
